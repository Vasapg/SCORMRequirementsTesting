# SCORMRequirementsTesting
This project is an exercise implemented following SCORM standards, which test the ability of a student to, given a document, identify the requirements in it and relate them to an action, rule, or state.

## How to implement an exercise
In order to add an exercise to the SCORM package, you must create an xml file that defines said exercise, following the sample xml in the exercises directory. Once the exercise is done, it must be uploaded to a git repository that is accessible for the SCORM package.
