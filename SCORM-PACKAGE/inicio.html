This assignment aims to exercise your ability to identify relations betweens four differents types of elements

Instructions

  1. In this exercices you will have to select with the cursor a phrase of the text which you consider to be whether a flow, depot, process or entity. Once selected you will be able to click on the button above the boxes to relate that element with a type.

  2. Once all elements are identified, yoy might advance to the next page, where you will have to create all the relations between the elements selected before. At any time you will be able to delete any relation created by clicking on it


  3. At any time, you might advance to the next page to see your model build, and be able to finish the exercise.


  4. Finally, you might compare your exercise with the solution, and advance to the next one. When all exercies are done, click on finish exercise to see the global grade and end the activity,

  5. At any point, before finishing the exercise, you will be able to navigate between the pages.
